## Basic Ready Rosie video library client created with Gatsby

1.  **Setup guide**

    1. Clone this repo
    2. run ```npm install```
    3. set correct api keys in gatsby-config.js
    4. run ```gatsby develop```
    5. run Ready Rosie rails app that serves Videos via graphql api
